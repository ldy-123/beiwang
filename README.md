# 备忘录

一个轻量的移动端备忘录 PWA，可安装到手机桌面离线使用，无需服务器。

## 功能

- 新建备忘录，支持标题、待办清单、标签、颜色标记
- 待办项支持截止时间提醒、拖拽排序、层级缩进
- 标签筛选、全文搜索
- 左滑卡片快速归档或删除
- 归档管理，支持一键恢复
- 导出 / 导入 JSON 备份，换机不丢数据
- 支持 PWA 安装，离线可用

## 使用

直接用浏览器打开 `index.html` 即可，无需构建。

部署到任意静态托管（GitHub Pages、Vercel 等）后，在手机浏览器访问，通过「添加到主屏幕」安装。

## 数据备份

数据默认存储在浏览器 `localStorage`，清除浏览器数据会丢失。

换机时：点击右上角 **⋮** → **导出备忘录** → 生成 `.json` 文件 → 在新设备导入恢复。

## 技术栈

纯原生 HTML / CSS / JavaScript，无任何依赖框架。
