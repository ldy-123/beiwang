# 备忘录

一个轻量的移动端备忘录 PWA，可安装到手机桌面离线使用，无需服务器。

## 产品背景与定位

市面上记录类应用存在两极分化：要么功能过于单一（如幕布、flomo），要么 All-in-one（如印象笔记）导致操作冗余。备忘录定位在两者之间——以**轻便记录**为核心，降低记录成本，将碎片化需求（灵感闪现、临时待办、学习心得、习惯打卡）统一到同一款应用中，避免频繁切换工具。

## 功能

### 备忘

- 新建备忘，支持标题、描述、待办清单、标签
- 四象限优先级，卡片标题前小圆点标识，自动排序
- 待办项支持截止时间提醒、拖拽排序、层级缩进
- 到期待办弹送系统通知
- 标签筛选、全文搜索、标签栏支持长按拖拽排序
- 左滑卡片快速归档或删除
- 卡片描述超出时截断显示，点击展开/收起

### 小记

- 富文本文档，支持加粗、下划线、文字颜色、插入图片
- 点击卡片展开/收起全文，长按或右上角 ⋯ 菜单进入编辑
- 左滑支持置顶、归档、删除
- 置顶小记在标签页内优先展示，多条按最新置顶排序
- 标签筛选

### 打卡

- 习惯追踪，32 种 emoji 图标可选
- 周视图 7 天圆点，点击即打卡
- 连续天数统计，🔥 badge 激励
- 左滑归档

### 归档

- 备忘 / 小记 / 打卡 三个子 tab 分类查看
- 支持一键恢复

### 数据

- 导出/导入全量 JSON 备份，换机不丢数据
- 移动端走系统分享面板，支持 AirDrop、微信等渠道
- 支持从剪贴板导入备份文本

## 使用

直接用浏览器打开 `index.html` 即可，无需构建。

部署到任意静态托管（GitHub Pages、Vercel 等）后，在手机浏览器访问，通过「添加到主屏幕」安装。

## 数据备份

数据默认存储在浏览器 `localStorage`，清除浏览器数据会丢失。

换机时：点击右上角 **⋮** → **导出备忘录** → 分享到微信/AirDrop → 在新设备导入恢复。

## 技术栈

纯原生 HTML / CSS / JavaScript，无任何依赖框架。
